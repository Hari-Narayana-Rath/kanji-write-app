# Kanji Write App

Phone-first kanji writing practice app with:

- stroke-order validation
- Anki-style spaced repetition
- separate `N5`, `N4`, and `N3` decks
- local profile and progress storage

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Data Sync

Regenerate the JLPT kanji deck:

```bash
npm run sync:deck
```
